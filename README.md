مشروع blackiris هو منصة إلكترونية تتيح لك مشاركة المنشورات المتعلقة بالنباتات وكتابة التعليقات التفاعلية. يضم الموقع نموذج الذكاء الاصطناعي مخصص لمساعدتك في الإجابة عن تساؤلاتك الزراعية، بالإضافة الى توفير بيئة اجتماعية تتيح لك المحادثة وتبادل الخبرات مع الآخرين. يهدف المشروع إلى بناء مجتمع متكامل يهتم بالنبات وتطوير المعرفة الخضراء لدى جميع المستخدمين حول العالم.
